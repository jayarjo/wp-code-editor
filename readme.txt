Adds proper code editor to all code editing textareas, including the Text tab of post edit screens

== Description ==

It is pain to edit plugin and theme files in WordPress backend. I doubt that anyone ever used it seriously at all. WP Code Editor adds [code editor](http://ace.c9.io/#nav=about) to all code editing screens, including the Text tab on posts edit pages.

== Installation ==

1. Go to: Plugins > Add New > upload
2. Browse for `wp-code-editor-x.x.x.zip` file
3. Hit `Install Now`
4. Activate

or

1. Upload contents of `wp-code-editor-x.x.x.zip` to `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
